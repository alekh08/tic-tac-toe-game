# 🎮 Multiplayer Tic Tac Toe

A real-time multiplayer Tic Tac Toe game built with React, Node.js, and Socket.io. Play with friends instantly!

## ✨ Features

- 🎯 **Real-time Multiplayer** - Two players compete instantly
- 👥 **Spectator Mode** - Multiple people can watch the game
- 🎨 **Beautiful UI** - Modern gradient design with smooth animations
- 📱 **Responsive** - Works perfectly on desktop and mobile
- 🏠 **Room System** - Share room IDs to play with friends
- ⚡ **Fast & Smooth** - WebSocket-based real-time updates
- 🎲 **Fair Play** - Turn-based gameplay with validation
